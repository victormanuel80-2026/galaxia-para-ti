# 💫 Mi Galaxia Para Ti

Una galaxia interactiva y romántica hecha con amor. ✨

## 🌌 Características

- ⭐ 700+ estrellas que parpadean y se mueven
- 🌠 Cometas cruzando el cielo
- ☄️ **Meteoritos con cartas de amor** — ¡tócalos para leer un mensaje!
- 💫 Rastro de estrellas siguiendo el dedo/cursor
- 💥 Botón de explosión de amor
- 📱 Funciona en celular y computadora

## 💌 Instrucciones

1. Abre `galaxia.html` en tu navegador
2. Espera que aparezcan los meteoritos ☄️ con el emoji 💌
3. Toca el meteorito para abrir una carta de amor
4. ¡Hay 8 cartas distintas!
